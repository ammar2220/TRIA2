# خطوات تفعيل الحفظ المشترك بين الأجهزة (عن طريق Upstash)

ملحوظة: "Vercel KV" القديمة اتلغت، ودلوقتي بتتعمل عن طريق **Upstash** اللي موجودة في نفس صفحة الـ Marketplace اللي انت واخد منها Screenshot. نفس الفكرة بالظبط، بس الاسم اتغير.

## 1. ارفع الملفات على GitHub
حدّث الريبو بتاعك `Calculatorforexpensive` بحيث يبقى شكل الملفات كده:

```
Calculatorforexpensive/
├── index.html          (النسخة الجديدة)
├── package.json        (جديد)
└── api/
    └── data.js          (جديد)
```

استبدل `index.html` القديم بالنسخة الجديدة، وضيف `package.json` في الجذر (نفس مكان `index.html`)، وضيف مجلد `api` وجواه `data.js`.

## 2. اربط الريبو بـ Vercel (لو مش مربوط بالفعل)
- ادخل على vercel.com وسجل دخول بحساب GitHub
- Add New → Project → اختار الريبو `Calculatorforexpensive` → Deploy

## 3. أضف قاعدة بيانات Upstash (Redis)
- من صفحة المشروع في Vercel، روح لتبويب **Storage**
- في قائمة **Marketplace Database Providers** دور على **Upstash** واضغط عليها (هي مكتوب جنبها "Serverless DB (Redis, Vector, Queue, Search)")
- اختار المنتج **Redis**
- اديها اسم (مثلاً `wallet-db`) واعمل Create
- بعد الإنشاء، Vercel هيسألك تربطها بمشروعك (Connect Project) — اختار مشروع `Calculatorforexpensive` واعمل Connect
- ده هيضيف تلقائياً الـ Environment Variables اللي محتاجها الكود — مش محتاج تكتب حاجة يدوي

## 4. أعد النشر (Redeploy)
بعد ما تربط Upstash، لازم تعمل Redeploy مرة واحدة عشان الـ Environment Variables تتفعل:
- من تبويب **Deployments** في المشروع
- اضغط على أحدث Deployment
- من الثلاث نقط (⋮) اختار **Redeploy**

## 5. جرب
- افتح اللينك بتاع الموقع من أي جهاز (كمبيوتر أو موبايل)
- سجل دخول وضيف حساب أو عملية
- افتحه من جهاز تاني بنفس اللينك — هتلاقي نفس الداتا ظاهرة

---

## ملاحظات مهمة

**الأمان**: الـ API الحالية (`/api/data`) مفتوحة لأي حد يعرف رابط موقعك (مفيش تحقق من هوية على مستوى الـ API نفسها، غير شاشة تسجيل الدخول اللي هي بس واجهة). ده مقبول لموقع شخصي بسيط، بس لو حابب تأمين إضافي قولّي وأضيفها.

**localStorage لسه موجود**: سبت نسخة احتياطية في `localStorage` كـ fallback، فلو النت وقع لحظة الحفظ، هتلاقي تنبيه على الشاشة، والداتا هتتخزن محلياً مؤقتاً لحد ما النت يرجع وتحفظ تاني.

**التكلفة**: باقة Upstash المجانية (Free tier) هتكفي بمراحل لاستخدام شخصي زي ده.
